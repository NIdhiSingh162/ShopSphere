# ShopSphere 🛒

A full-stack E-Commerce Web Application built using React, TypeScript, Node.js, Express.js, and MySQL. ShopSphere allows users to browse products, manage carts, place orders, and provides an admin dashboard for product and order management.

## Features

* User Registration & Login
* Product Listing
* Product Search
* Product Details Page
* Shopping Cart
* Admin Dashboard
* Add Products
* Edit Products
* Delete Products
* Image Upload
* Order Management
* Responsive UI

## Tech Stack

### Frontend

* React
* TypeScript
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Installation

### Clone Repository

```bash
git clone https://github.com/NIdhiSingh162/ShopSphere.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```
